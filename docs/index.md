# NAT Docs Index

Welcome to the NAT Docs! This documentation serves as a comprehensive guide for all things related to NAT Indonesia and its initiatives. Dive into our detailed documentation to learn more about our systems, standards, and practices.


# Technology Stack

- **Frontend**: HTML, CSS, JavaScript, TypeScript, Vue.js, Tailwind CSS, Next JS, Alpine.js
- **Backend**: PHP, Laravel, Go, .NET ASP, Node.js
- **Database**: MySQL, MariaDB
- **API**: RESTful API, JSON
- **Deployment**: Docker, CPanel, Ubuntu, Apache
- **Monitoring**: Uptime Kuma, Sentry, Dependabot
- **CI/CD**: Jenkins, GitHub Actions, SonarQube
- **Testing**: PHPUnit, Postman
- **Documentation**: MkDocs, Material for MkDocs, Markdown
- **Design**: Figma
- **Hosting/SaaS**: Azure, Cloudflare, Firebase
- **Others**: OpenAI, Home Assistant, WordPress, Visual Studio Code
